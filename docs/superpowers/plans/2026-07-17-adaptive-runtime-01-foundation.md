# Adaptive Runtime Execution Foundation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Establish a baseline-compatible runtime switch, immutable model policy, audited provider gateway, and supervised credential-free Agent process that works from source and packaged binaries.

**Architecture:** The CLI performs an early branch: baseline retains the existing SDK/legacy Session path, while adaptive creates a durable Task and calls `AdaptiveController`. The Controller owns model resolution and provider credentials; an Agent child communicates through bounded NDJSON frames over stdio and references a Controller-created ContextManifest rather than sending arbitrary model identity or prompt history.

**Tech Stack:** TypeScript, Effect services/LayerNode, Drizzle SQLite migrations, `@opencode-ai/llm`, OpenCode LocationServiceMap, Effect ChildProcessSpawner, yargs, Bun test/build.

```ts
export * as AdaptiveModelPolicy from "./model-policy"

import { createHash } from "node:crypto"
import { AdaptiveTask } from "@opencode-ai/schema/adaptive-task"

export type Input = Omit<AdaptiveTask.ModelPolicy, "hash">

const canonical = (input: Input) =>
  JSON.stringify({
    providerID: input.providerID,
    modelID: input.modelID,
    ...(input.variant === undefined ? {} : { variant: input.variant }),
    effectiveContextLimit: input.effectiveContextLimit,
    outputReserve: input.outputReserve,
    safetyReserve: input.safetyReserve,
  })

export const create = (input: Input) =>
  new AdaptiveTask.ModelPolicy({
    ...input,
    hash: `sha256:${createHash("sha256").update(canonical(input)).digest("hex")}`,
  })

export function assertEqual(expected: AdaptiveTask.ModelPolicy, actual: AdaptiveTask.ModelPolicy) {
  if (expected.hash === actual.hash && canonical(expected) === canonical(actual)) return
  throw new Error(`Adaptive ModelPolicy mismatch: expected ${expected.hash}, received ${actual.hash}`)
}
```

export interface Interface {
  readonly admit: (input: AdmissionInput) => Effect.Effect<Admission, AdmissionError>
  readonly streaming: (requestID: AdaptiveTask.RequestID) => Effect.Effect<void>
  readonly settle: (input: SettlementInput) => Effect.Effect<void>

}

export type ValidityProof =
  | { readonly valid: true; readonly providerID: string; readonly modelID: string; readonly policyHash: string; readonly requests: number }
  | { readonly valid: false; readonly code: "INVALID_MODEL_MIXING"; readonly reasons: readonly string[]; readonly requests: number }
